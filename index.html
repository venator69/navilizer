How to play
  • New terrain builds a random 16×16 map with walls (~22%). Start is top-left (blue), goal bottom-right (green). A valid path always exists.
  • Pick BFS, DFS, or A* and press Run to see the graph (gray edges, nodes) and exploration highlights, then the forklift following the found path (BFS and A* are shortest here; DFS may be longer).
  • Q-learning updates tabular Q each episode (heatmaps). Local best is the smallest number of environment steps in an episode that reached the goal (updated whenever a shorter successful episode appears). Checkbox Final ep path only: when off (default), the rollout animates whenever the goal is reached each episode; when on, rollout animation plays only after the last episode.
  • Q-network: With Show data acquisition viz on, each rollout animates (orange trace), and logs list the replay-buffer tail (transition fields). With it off, phase 1 is text-only; after training we show heatmaps and the greedy path. Local best is the minimum rollout steps among acquisition episodes that reached the goal. Phase 2 is TD replay on the buffer regardless.
  • Stop cancels a long run. Serve this folder with any static server (or open via index.html if your browser allows module scripts locally).
